# Purpose

This document defines the guiding philosophy of the Art Framework.

It establishes the principles that govern every artistic and technical decision made within the framework.

These principles are intended to remain stable across campaigns and provide a consistent foundation for future artwork.

---

# Core Philosophy

The objective of this framework is to create artwork that feels cohesive, intentional, and timeless.

Every image should appear as though it belongs to the same professionally illustrated collection, regardless of when it was created or which campaign it supports.

Consistency is valued over novelty.

Visual identity is more important than individual experimentation.

The framework exists to reduce interpretation, minimize artistic drift, and produce repeatable results over long-term projects.

---

# Guiding Principles

## Consistency First

Every approved image becomes part of a larger collection.

New artwork should reinforce the existing visual identity rather than redefine it.

---

## Intentional Design

Every visual decision should have a purpose.

Composition, lighting, color, magic, equipment, and decorative elements should contribute to storytelling.

Visual clutter should always be avoided.

---

## Quality Over Quantity

The objective is not to generate many images.

The objective is to create artwork worthy of becoming part of the permanent collection.

Generation may be repeated as many times as necessary to achieve the desired quality.

---

## Preserve Canon

Once artwork, character information, or framework documentation is approved, it becomes canonical.

Nothing should be modified without explicit approval.

This principle exists to prevent artistic drift throughout the life of a campaign.

---

## Framework Before Campaign

The framework defines **how** artwork is created.

Campaigns define **what** artwork is created.

Campaign-specific decisions should never permanently alter the framework unless explicitly approved.

---

## Campaign Independence

The framework is designed to support multiple campaigns.

Fantasy, science fiction, horror, whimsical, historical, or any future setting should be able to use the same framework while documenting only the campaign-specific differences.

---

# Design Philosophy

Artwork should feel:

- Intentional
- Immersive
- Cohesive
- Believable
- Cinematic
- Timeless

The viewer should remember the collection as a whole rather than noticing individual stylistic inconsistencies.

---

# Decision Standard

When multiple artistic options are equally valid, choose the option that best supports:

1. Consistency
2. Canonical Accuracy
3. Readability
4. Storytelling
5. Visual Quality

Novelty should never take priority over consistency.

---

# Framework Integrity

Every rule within the framework should exist in exactly one location.

Documentation should avoid duplication whenever possible.

If two documents appear to conflict, the higher-authority document takes precedence.
