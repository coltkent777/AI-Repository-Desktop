# Purpose

This document defines the recurring visual language shared by artwork created using the Art Framework.

Visual Language establishes the artistic vocabulary that gives every collection a recognizable identity while allowing each campaign to define its own unique symbols and themes.

Campaigns may expand this language but should not replace the framework without explicit approval.

---

# Visual Language Standards

## Campaign Identity

Every campaign must establish its own recurring visual motifs.

These motifs become part of the campaign's identity and should appear consistently throughout its artwork.

Examples include, but are not limited to:

- Hidden symbols
- Shared magical effects
- Recurring colors
- Signature creatures
- Common decorative elements
- Story-specific visual references

Campaign motifs are defined within that campaign's documentation and are not part of the framework itself.

---

## Character Manifestations

Character Manifestations remain the primary method of communicating identity.

They should remain visually connected to the character and reinforce the subject without overwhelming the composition.

The character should always remain the focal point.

---

## Magical Language

Magic should possess a recognizable visual language.

Recurring elements are encouraged, including:

- Arcane tracery
- Ancient runes
- Magical glyphs
- Flowing magical geometry
- Energy patterns

Magic should appear intelligent and intentionally constructed rather than random visual effects.

---

## Ornamentation

Decorative elements should combine:

- Elegance
- Antiquity
- Organic forms
- Geometric structure
- Minimal restraint

Decoration should support storytelling and visual identity rather than exist solely for complexity.

---

## Visual Density

Artwork should feel balanced while leaning toward rich visual detail.

Every element should contribute to the composition.

Visual clutter should always be avoided.

---

## Repetition

Consistent visual motifs are encouraged.

Examples include:

- Shared rune styles
- Similar magical construction
- Divine symbolism
- Nature motifs
- Shadow language
- Cultural ornamentation

Consistency strengthens the visual identity of a campaign.

---

## Signature Collection Identity

Regardless of campaign setting, artwork created using this framework should be recognizable through its consistent visual language.

The framework's signature characteristics include:

- Character-focused composition
- Character Manifestations
- Magical tracery
- Arcane runes or glyphs
- Rich painterly rendering
- Black or transparent backgrounds
- Fantasy lighting

Campaign-specific motifs are added to these standards rather than replacing them.

---

# Campaign Responsibilities

Each campaign should define:

- Signature motifs
- Hidden recurring elements
- Shared symbols
- Visual themes
- Unique magical language

These become part of the campaign's artistic identity.

Future campaign setup should explicitly include defining these elements before artwork generation begins.

---

# Campaign Exceptions

Campaign-specific visual language supplements the framework.

It does not replace the framework unless explicitly approved.
