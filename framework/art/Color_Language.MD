# Purpose

This document defines the permanent color standards for artwork created using the Art Framework.

Color exists to strengthen character identity, improve readability, and maintain visual consistency across every campaign.

Campaigns and characters define their own palettes within these framework standards.

---

# Color Standards

## Framework Color

Aged Silver is the permanent framework color.

It should appear consistently throughout artwork whenever appropriate, including but not limited to:

- Armor
- Ornamentation
- Weapons
- Jewelry
- Magical details
- Decorative accents
- Metallic surfaces

Aged Silver serves as the visual thread connecting every campaign created with this framework.

---

## Character Identity

Every character should have a signature color palette.

The signature palette is defined within that character's documentation.

This palette becomes the primary color language used for:

- Magic
- Clothing
- Ornamentation
- Equipment
- Character Manifestations
- Decorative elements

The signature palette reinforces the character's identity across all artwork.

---

## Saturation

Colors should be intentionally rich and vibrant.

Muted or desaturated palettes should only be used when intentionally supporting a campaign or character's narrative.

---

## Readability

Color should always improve readability.

Important subjects, equipment, and Character Manifestations should remain visually distinct from the background.

No important visual element should disappear due to insufficient contrast.

---

## Magical Color

Magic should reinforce the character's identity.

Magical effects should consistently reflect the character's established color palette rather than using arbitrary colors.

Consistent magical color language strengthens recognition and visual storytelling.

---

## Color Harmony

Artwork should feel visually cohesive.

Supporting colors should complement the character's signature palette without competing for attention.

The signature color should remain dominant.

---

## Campaign Responsibility

Campaign documentation establishes any shared campaign colors beyond the permanent framework standards.

Character documentation establishes each character's signature palette.

The framework defines how color is used.

Campaigns and characters define which colors are used.

---

# Campaign Exceptions

Campaigns may introduce additional shared colors when appropriate.

The permanent use of Aged Silver remains unchanged.
