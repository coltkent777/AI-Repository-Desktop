# Framework README

> **Purpose**
>
> This document defines the artistic standards and visual philosophy
> that every piece of artwork in this repository should follow.
>
> These rules establish a consistent visual language across all
> campaigns while allowing each campaign and character to retain a
> unique identity.

------------------------------------------------------------------------

# Framework Philosophy

The framework exists to create artwork that is recognizable as part of
the same collection.

Consistency should come from shared artistic principles---not identical
subjects.

Every image should feel as though it belongs in the same world,
regardless of the character or campaign.

------------------------------------------------------------------------

# Artistic Philosophy

Artwork should strive for:

-   Epic high fantasy
-   Elegant restraint
-   Rich visual storytelling
-   Believable environments
-   Lived-in realism
-   Strong atmosphere
-   Timeless quality

Complexity should enhance the artwork, never overwhelm it.

------------------------------------------------------------------------

# Global Visual Language

Every image should maintain consistency in:

-   Lighting
-   Rendering quality
-   Composition philosophy
-   Environmental realism
-   Material definition
-   Scale
-   Detail level

Campaign documents define the unique elements. The framework defines the
shared language.

------------------------------------------------------------------------

# Color Philosophy

Color should communicate identity.

General principles:

-   Every character has a primary signature color.
-   Supporting colors should reinforce---not compete with---the
    signature color.
-   Metallic accents should complement the palette.
-   Color should guide the viewer's eye naturally.

------------------------------------------------------------------------

# Composition Philosophy

Composition should support storytelling.

Images should prioritize:

-   Clear focal points
-   Balanced visual weight
-   Natural framing
-   Depth
-   Readability

Scene-specific composition is determined by the individual image
request.

------------------------------------------------------------------------

# Lighting Philosophy

Lighting should enhance mood while preserving visual clarity.

Lighting choices should feel believable within the world while
emphasizing the subject and reinforcing the emotional tone of the scene.

------------------------------------------------------------------------

# Rendering Philosophy

Artwork should aim for:

-   High-quality fantasy illustration
-   Rich material definition
-   Convincing textures
-   Natural depth
-   Cohesive environmental integration

Visual effects should support the artwork rather than dominate it.

------------------------------------------------------------------------

# Continuity

All approved artwork should feel connected through:

-   Shared artistic philosophy
-   Consistent rendering
-   Unified visual language
-   Respect for approved repository information

Campaigns define identity.

The framework defines consistency.

------------------------------------------------------------------------

# Relationship to Campaigns

The framework provides the global rules.

Campaign documents provide campaign-specific information.

If campaign information expands upon the framework without contradicting
it, the campaign takes precedence for that campaign only.

If a contradiction exists, ChatGPT should stop and ask the user before
proceeding.
