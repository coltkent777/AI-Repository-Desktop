# LOCATION_TEMPLATE

> **Purpose:** Document campaign-specific locations for consistent AI
> artwork.
>
> Locations are only used when explicitly requested. An approved
> location overrides the framework's default black background and
> becomes the canonical environment for artwork created in that setting.
>
> **Character placement and composition are determined by the individual
> image request and are not defined by this template.**
