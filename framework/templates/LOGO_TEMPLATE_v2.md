# LOGO_TEMPLATE

> **Purpose:** Define how ChatGPT should assemble a character-specific
> Dungeons & Dragons logo prompt from approved repository information.
>
> Logos are **derived assets**. They do not collect new character
> information. They are generated from the approved Character document
> together with the Framework and Campaign rules.

# Locked Base Logo

-   Official Dungeons & Dragons logo
-   Preserve the words "Dungeons & Dragons"
-   Preserve the typography whenever possible
-   Preserve the official ampersand
-   Black background unless explicitly overridden by the framework

------------------------------------------------------------------------

# Prompt Assembly

ChatGPT should infer the logo's design from the approved repository
information, including:

-   Race
-   Class
-   Subclass
-   Primary color
-   Equipment
-   Visual design language
-   Symbols and motifs
-   Personality and visual presence
-   Hidden storytelling opportunities

No new character facts should be invented.

------------------------------------------------------------------------

# Required Design Goals

-   Character's primary color is represented.
-   Elegant restraint.
-   High fantasy ornamentation.
-   Hidden symbolic storytelling.
-   Consistent with the framework's artistic philosophy.
